## 第 1 步：安装 Docker Desktop

### 1.1 下载

访问：**https://www.docker.com/products/docker-desktop/**

点击 **"Download for Windows"** 下载安装程序。

### 1.2 安装

1. 双击安装程序 `Docker Desktop Installer.exe`
2. 勾选 **"Use WSL 2 instead of Hyper-V"**（推荐）
3. 点击 **OK** 开始安装
4. 安装完成后点击 **Close and restart**（重启电脑）

### 1.3 验证

重启后，打开终端（PowerShell 或 Git Bash），输入：

```powershell
docker --version
docker compose version
```

预期输出类似：

```
Docker version 28.x.x
Docker Compose version v2.x.x
```

### 1.4 启动 Docker Desktop

- 从开始菜单搜索 **Docker Desktop** 并启动
- 等待右下角系统托盘 Docker 图标停止动画、变稳定
- 可以在终端运行 `docker ps` 确认无报错

### 9.2 创建 seed 脚本

用编辑器创建 `E:\Code_development\Coffee_bake_system\backend\scripts\seed_standard_terms.py`，内容如下：

```python
"""Seed standard terms — idempotent. Safe to run multiple times."""
import asyncio
import sys
from pathlib import Path

# Ensure backend is on path
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))

from sqlalchemy import select
from sqlalchemy.ext.asyncio import create_async_engine, AsyncSession
from app.core.config import settings
from app.models.all_models import StandardTerm

SEED_DATA = {
    "variety": ["铁皮卡", "波旁", "卡杜拉", "卡杜艾", "瑰夏", "SL28", "SL34", "帕卡马拉", "埃塞俄比亚原生种", "卡蒂姆"],
    "process": ["水洗", "日晒", "蜜处理", "厌氧发酵", "半水洗", "湿刨法"],
    "region": ["非洲", "中美洲", "南美洲", "亚洲", "大洋洲"],
    "flavor": [
        "花香", "柑橘", "莓果", "巧克力", "坚果", "焦糖", "蜂蜜", "红酒",
        "核果", "香料", "烟草", "草本", "发酵", "奶油", "烤面包", "黑莓",
        "柠檬", "橙子", "葡萄柚", "百香果", "芒果", "桃子", "杏子", "椰子",
        "杏仁", "核桃", "榛子", "黑巧克力", "牛奶巧克力", "太妃糖",
        "茉莉花", "玫瑰花", "薰衣草", "伯爵茶", "绿茶",
    ],
    "roast_level": ["极浅烘焙", "浅烘焙", "中浅烘焙", "中度烘焙", "中深烘焙", "深度烘焙", "极深烘焙"],
    "brew_method": ["手冲", "法压壶", "爱乐压", "虹吸壶", "摩卡壶", "意式浓缩", "冷萃", "冰滴"],
    "drink_temperature": ["热", "冷"],
    "drink_form": ["黑咖啡", "加奶", "其他"],
    "evaluator_type": ["烘焙师", "同事", "顾客"],
    "supplier": ["供应商A", "供应商B", "淘宝", "微批次直采"],
}


async def seed(db: AsyncSession) -> None:
    for category, values in SEED_DATA.items():
        for order, value in enumerate(values):
            # Check existence
            result = await db.execute(
                select(StandardTerm).where(
                    StandardTerm.category == category,
                    StandardTerm.value == value,
                )
            )
            existing = result.scalar_one_or_none()
            if existing:
                print(f"  [SKIP] {category}/{value} — 已存在")
                continue

            term = StandardTerm(
                category=category,
                value=value,
                display_order=order,
                is_active=True,
            )
            db.add(term)
            print(f"  [OK]   {category}/{value}")
    await db.flush()
    print(f"\n标准词表 Seed 完成。")


async def main() -> None:
    engine = create_async_engine(settings.database_url)
    async with AsyncSession(engine) as db:
        await seed(db)
        await db.commit()
    await engine.dispose()


if __name__ == "__main__":
    asyncio.run(main())
```

## 常见问题排查

### Docker 启动失败

```powershell
# 检查 WSL 是否已安装
wsl --install

# 如果 Docker 一直转圈，尝试重启 Docker 服务
net stop com.docker.service
net start com.docker.service
```

### 端口被占用

```powershell
# 检查 5432（PostgreSQL）或 8000（后端）端口
netstat -ano | findstr :5432
netstat -ano | findstr :8000
```

### 数据库连接失败

确认：
1. Docker Desktop 正在运行
2. `docker compose ps` 显示 `healthy`
3. `.env` 中的 `DATABASE_URL` 使用 `localhost`（不是 `db`）

### Alembic 找不到模型

确认 `alembic.ini` 中有 `prepend_sys_path = .`（句点，表示当前目录）。

### Token 过期

`.env` 中 `ACCESS_TOKEN_EXPIRE_MINUTES=1440`（24小时），重启后端需重新登录。
